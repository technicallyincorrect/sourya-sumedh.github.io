# Sourya Sumedh Portfolio

A static GitHub Pages portfolio for Sourya Sumedh Bandi.

## Files

- `index.html` — portfolio content
- `style.css` — styling
- `script.js` — mobile navigation and current year
- `resume.pdf` — downloadable resume

## Publish on GitHub Pages

1. Create or open the repository named `sourya-sumedh.github.io`.
2. Upload all files from this folder to the repository root.
3. Open **Settings → Pages**.
4. Set **Source** to `Deploy from a branch`.
5. Select `main` and `/(root)`.
6. Save and wait a few minutes.
7. Open `https://sourya-sumedh.github.io`.
